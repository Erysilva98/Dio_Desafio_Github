Projeto Prático - Criando um Respositório no GitHub


Detalhes do Projeto :


      Criando Respositório no GitHub
      Realizando o Git clone ( Cópia do Respositório na máquina local )
      Modificando os Arquivos na máquina local, ( Criar novos, Editar, Deletar e etc. )
      Gerenciando os Arquivos com o Git
      Realizando um git status ( Verificar o status dos Arquivos )
      Git add . ( Adicionado os Arquivos e alterações na ramificação do Git)
      Criando Commit na Branch Main ( git commit -m "Descrição")
      Verificando se todos os arquivos estão prontos para envio ( git status )
      Enviando os Arquivos da máquina local para o GitHub ( git push origin main )
